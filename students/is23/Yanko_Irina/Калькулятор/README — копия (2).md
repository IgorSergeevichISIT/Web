**Лабораторная работа №2: Реализация логики калькулятора на TypeScript**

### Краткое описание
Проект развивает интерфейс, созданный в ЛР №1, путем добавления интерактивной логики на **TypeScript**. Было выполнено 12 заданий, включая реализацию сложных математических операций, накапливаемых вычислений и профессиональной обработки критических ошибок.

### Содержание проекта и структура файлов
*   `dist/` — содержит скомпилированный JavaScript-файл `main.js` (или `script.js`), готовый к исполнению в браузере.
*   `src/` — содержит исходные файлы на Node.js, включая основной скрипт `main.ts` (или `script.ts`) с логикой калькулятора.
*   Остальные файлы (такие как `index.html` и `style.css`) являются сопроводительными, часть из них повторяет или развивает структуру из ЛР №1 для обеспечения работы интерфейса.

### Детализация файлов
| Файл | Назначение |
| :--- | :--- |
| `index.html` | Обновленная верстка калькулятора из ЛР №1 с добавлением новых кнопок для расширенного функционала. Подключает скомпилированный скрипт `main.js`. |
| `style.css` | Файл стилей из первой работы, дополненный для оформления новых элементов управления. |
| `main.ts` | **Исходный код на TypeScript.** Содержит класс `Calculator` с реализацией всей логики, сопровождаемый подробными комментариями по соответствию каждому из 12 заданий. |
| `main.js` | **Скомпилированный код на JavaScript.** Файл, который непосредственно выполняется браузером и подключается к `index.html`. |

### Инструкция по запуску
Для корректной работы калькулятора необходимо обеспечить загрузку скомпилированного файла `main.js` через локальный веб-сервер.

1.  **Установите** расширение **Live Server** в Visual Studio Code.
2.  **Откройте** корневую папку проекта в VS Code.
3.  **Кликните** правой кнопкой мыши по файлу `index.html` и выберите **"Open with Live Server"** (или нажмите кнопку **"Go Live"** в правом нижнем углу VS Code).
4.  Приложение откроется в браузере, и весь заявленный функционал (12 заданий) будет доступен.

### Реализованный функционал 
В файле `main.ts` реализованы следующие операции и возможности:

*   **Базовые арифметические функции:** Смена знака числа (+/-), вычисление процента (%), удаление последней цифры (Backspace).
*   **Расширенные математические операции:** Вычисление квадратного корня (√), возведение в квадрат (x²), факториал (x!), добавление трех нулей (000).
*   **Индивидуальная операция:** Вычисление логарифма по основанию 2 ($\log_2(x)$).
*   **Интерактивные элементы:** Динамическая смена цвета фона и дисплея калькулятора по нажатию кнопки.
*   **Обработка ошибок:** Защита от некорректных операций с блокировкой ввода после возникновения критических ошибок (таких как `Infinity`, `Error`, `NaN`).
