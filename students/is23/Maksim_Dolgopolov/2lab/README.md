# Отчёт по лабораторной работе №2
  
**Калькулятор на JavaScript**

---

## Цель работы

Целью данной работы было изучение основ языка JavaScript для создания интерактивных веб-интерфейсов на примере разработки функционального калькулятора. В ходе выполнения предстояло добавить логику работы кнопок, расширить функционал базовых операций и реализовать дополнительные математические операции, а также интерактивные элементы управления.

---

## Что сделано

Был создан файл `calc_script.js`, в котором реализован класс `Calculator`, управляющий всеми состояниями и операциями. Калькулятор поддерживает основные арифметические действия (сложение, вычитание, умножение, деление), а также дополнительные операции: смена знака, вычисление процента, квадрат числа, квадратный корень, факториал, десятичный логарифм и добавление трёх нулей к числу.

Реализована обработка ввода цифр и точки, управление состоянием ввода через флаги, предотвращение ошибок (деление на ноль, корень из отрицательного числа, факториал нецелого числа). Добавлена кнопка для удаления последней цифры (backspace).

Также была реализована смена темы интерфейса (светлая/тёмная) по кнопке «Mode». При переключении изменяются цвета фона калькулятора, кнопок и поля вывода.

Верстка была обновлена: поле вывода заменено на `<input readonly>`, добавлены новые кнопки для расширенного функционала, использована CSS Grid для расположения элементов. Все кнопки получили соответствующие обработчики событий через единый механизм инициализации в классе `Calculator`.

---

## Итог

В результате создан полностью рабочий калькулятор с расширенным набором операций, интерактивным переключением темы и обработкой ошибочных ситуаций, демонстрирующий применение JavaScript для создания динамического пользовательского интерфейса.
