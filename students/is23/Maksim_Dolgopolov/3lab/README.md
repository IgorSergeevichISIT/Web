# Отчёт по лабораторной работе №3  
**Простое веб-приложение. Верстка**

---

## Цель работы

Целью работы было изучение основ построения веб-приложений с использованием Node.js, npm и модульной архитектуры на JavaScript. В ходе выполнения предстояло создать модульное приложение с использованием Bootstrap, реализовать страницу с интерактивным выбором темы и динамической загрузкой изображений.

---

## Что сделано

Было инициализировано npm-приложение, подключён Bootstrap для стилизации. Создана модульная структура проекта с разделением на страницы и компоненты. Реализована главная страница (`MainPage`), содержащая компонент `ButtonGroup` — группу кнопок-выпадающих списков для выбора времени года и времени суток (день/ночь).

Каждая кнопка в группе соответствует одному из восьми вариантов (4 сезона × 2 времени суток) и содержит `data-id`, по которому определяется выбранное изображение. При клике на любой вариант вызывается метод `clickPic`, который динамически создаёт и отображает компонент `Image` с соответствующей фотографией.

Все изображения хранятся в виде массива данных внутри класса `Image`, каждый элемент содержит `id` и ссылку `src`. Для стилизации использован кастомный CSS-файл `bs-edit.css`, в котором заданы цвета выпадающих меню, размеры кнопок и внешний вид страницы.

---

## Итог

В результате разработано одностраничное веб-приложение с интерактивным интерфейсом, позволяющее выбирать изображение по теме «Времена года» через группу кнопок Bootstrap. Приложение демонстрирует работу с модулями ES6, динамической вставкой HTML, обработкой событий и адаптивной стилизацией.
