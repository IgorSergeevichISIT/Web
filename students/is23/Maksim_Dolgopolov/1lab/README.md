# Отчёт по лабораторной работе №1
  
**Калькулятор на HTML/CSS**

---

## Цель работы

Целью работы было изучение основ HTML и CSS через создание и стилизацию веб-страницы калькулятора. В ходе выполнения работы предстояло ознакомиться с кодом готовой верстки и выполнить ряд заданий по изменению внешнего вида и добавлению новых элементов интерфейса.

---

## Что сделано

Была изменена цветовая гамма калькулятора: фон страницы получил изображение, сам калькулятор — тёмно-синий фон, кнопки стали красными, а операторы — синими. Кнопки выполнены квадратными, поле вывода — с закруглёнными краями и белым фоном. Шрифт заменён на Agency FB, для разных типов кнопок настроена разная толщина текста.

Добавлены интерактивные элементы: кнопка Mode для смены темы, кнопка Git со ссылкой на GitHub, а также сворачивающийся блок с информацией об авторе. Страница выровнена по центру, увеличен размер поля вывода, настроены эффекты при наведении на кнопки.

В текст цели работы добавлено форматирование: выделены курсивом, жирным и подчёркиванием ключевые слова — «знакомство», «HTML» и «CSS». Общая стилистика калькулятора разработана уникально, без повторения шаблонных решений.

---

## Итог

В итоге создана стилизованная веб-страница калькулятора, демонстрирующая применение базовых и продвинутых возможностей HTML и CSS, включая работу с цветами, шрифтами, отступами, сетками и интерактивными элементами.
