# Лабораторная работа №1: HTML + CSS для калькулятора

Цель: Создание вёрстки и стилизация интерфейса калькулятора с использованием HTML и CSS.

---

## Структура проекта

- `index.html` — структура калькулятора: панель вывода и кнопки
- `style.css` — стили для темного интерфейса, расположение элементов и внешний вид
- `readme.md` — описание лабораторной работы

---

## Как запустить

Откройте файл `index.html` в браузере — калькулятор отобразится сразу без дополнительной сборки.

---

## Примечание

- Все стили реализованы вручную, без фреймворков.
- Интерфейс соответствует требованиям: панель вывода сверху, кнопки сгруппированы по логическим блокам.
- Проект не требует JavaScript для отображения.
