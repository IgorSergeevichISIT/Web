# Лабораторная работа 2: JavaScript(TypeScript) + HTML + CSS

Цель: Изучение базовой структуры проекта на TypeScript с интеграцией в HTML и CSS.

---

## Структура проекта

- **src/** — папка с исходными файлами:
  - `script.ts` — основной TypeScript-код. Компилируется в JavaScript.
  - `package.json` — метаданные проекта, зависимости и npm-скрипты (например, build, start).
  - `package-lock.json` — фиксация точных версий всех зависимостей для воспроизводимости сборки.
- **index.html** — главная страница. Подключает скомпилированный JS и CSS.
- **style.css** — таблица стилей для оформления страницы.
- **readme.md** — текущий документ с описанием проекта.
- **tsconfig.json** — конфигурация компилятора TypeScript.

---

## Как запустить

1. Установите зависимости:
   npm install
2. Скомпилируйте TypeScript в JavaScript:
   npx tsc
   (Скомпилированный файл script.js появится в папке src/)
3. Откройте index.html в браузере - приложение готово к работе.

---

## Примечание

- Для автоматической перекомпиляции при изменении .ts файлов можно использовать npx tsc --watch.
- Все файлы должны быть в кодировке UTF-8 без BOM.
- Проект не требует сервера. Работает локально через file://.
