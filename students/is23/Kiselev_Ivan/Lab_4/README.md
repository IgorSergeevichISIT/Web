# Лабораторная работа 4: Взаимодействие с VK API через XMLHttpRequest

**Цель:** Реализация двухстраничного веб-приложения, взаимодействующего с внешним API (VK) с использованием `XMLHttpRequest`. Освоение архитектуры клиентского приложения с разделением на модули, компоненты и страницы.

---

## Структура проекта

- **modules/** — вспомогательные модули для работы с API:
  - `consts.js` — константы: `groupId`, `accessToken`, версия API
  - `urls.js` — генератор URL-адресов для методов VK API
  - `ajax.js` — обёртка над `XMLHttpRequest` для выполнения POST-запросов
- **components/** — переиспользуемые UI-компоненты:
  - `ProductCardComponent.js` — карточка участника группы
  - `ProductComponent.js` — детальная карточка пользователя
  - `BackButtonComponent.js` — кнопка «Назад»
- **pages/** — логика страниц приложения:
  - `MainPage.js` — главная страница с выбором сортировки и списком участников
  - `UserPage.js` — страница профиля выбранного пользователя
- **css/** — кастомные стили:
  - `style.css` — оформление в стиле ВКонтакте (голубо-синяя тема, центрирование)
- **index.html** — точка входа. Подключает Bootstrap и собственные стили
- **main.js** — инициализация приложения и маршрутизация
- **README.md** — текущий документ

Проект использует **нативные ES-модули** (`import`/`export`) и **не требует сборки** (например, через Webpack или TypeScript).

---

## Как запустить

1. **Убедитесь, что установлены**:

   - `Node.js `
   - Расширение `Live Server` в VS Code
2. **Настройте доступ к VK API**:

   - Создайте сообщество в ВКонтакте
   - Получите **ключ доступа сообщества** с правами: `groups`, `users`
   - Заполните файл `modules/consts.js`:
     ```js
     export const groupId = 'ВАШ_ЧИСЛОВОЙ_ID_ГРУППЫ';
     export const accessToken = 'ВАШ_КЛЮЧ_ДОСТУПА';
     export const version = '5.199';
     ```
3. **Запустите сервер**:

   - Откройте проект в VS Code
   - Нажмите **«Go Live»** в правом нижнем углу
     или
     ПКМ по `index.html` → **Open with Live Server**
4. **Обход CORS**:

   - Установите расширение, отключающее CORS (например, `CORS Unblock`)
   - Активируйте его на вкладке с приложением

> Веб-приложение откроется по адресу вида `http://127.0.0.1:5500`

## Реализованный функционал (вариант 1)

- Получение списка участников группы через `groups.getMembers`
- Сортировка по параметру `sort`:`id_asc`, `id_desc`, `time_asc`, `time_desc`
- Отображение карточек с аватаром, именем и фамилией
- Переход на страницу профиля по клику
- На странице пользователя отображаются:
  - Фото профиля
  - Полное имя
  - ID
  - Никнейм (в формате `@username`)
  - Дата рождения (если указана, иначе «Не указано»)
- Кнопка «Назад» для возврата к списку участников
- Интерфейс в стиле ВКонтакте (голубо-синяя тема)
- Полное центрирование контента на всех страницах

## Примечание

- Проект **работает только через локальный сервер** (Live Server). Запуск через `file://` не поддерживается.
- **Не коммитьте `accessToken`** в публичные репозитории — это конфиденциальные данные.
- Все файлы должны быть в кодировке **UTF-8 без BOM**.
- Если фото участника не отображается на главной странице — это ограничение VK API. Однако на странице пользователя оно почти всегда доступно.
- После завершения работы рекомендуется **отозвать ключ доступа** в настройках группы ВКонтакте.
