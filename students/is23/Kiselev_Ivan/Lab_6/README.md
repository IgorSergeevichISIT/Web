# Лабораторная работа №6. Реализация собственного API через свой backend

Локальный backend на **Node.js + Express**, заменяющий внешнее API (VK).
Фронтенд взаимодействует с сервером через `fetch`.

## Возможности

- Просмотр списка пользователей
- Добавление нового пользователя
- Удаление пользователя
- Сортировка по ID (возрастание / убывание)
- Автоматическая генерация аватара по инициалам
