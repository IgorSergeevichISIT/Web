## Структура проекта 

| Файл             | Назначение                                                                 |
|------------------|---------------------------------------------------------------------------|
| `calculator.html` | Основная HTML-структура калькулятора                                      |
| `style.css`       | Стили для тёмной/светлой темы, нейроморфный дизайн                        |


---

## Проделанная работа

**1. Семантическая HTML-разметка**  
   - Калькулятор разбит на логические блоки (цифры, операции, доп. элементы).
   - Использованы классы и ID для стилизации и дальнейшей работы с JavaScript.

**2. Стилизация CSS**  
   - Реализована тёмная тема по умолчанию и светлая тема по кнопке переключения.
   - Применён дизайн с градиентами, тенями и плавными переходами.
   - Все элементы центрированы на странице с использованием flexbox.
   - Используются фиксированные размеры в пикселях.

**3. Дополнительные элементы**  
   - Кнопка смены темы.
   - Ссылка на GitHub.
   - Выпадающий список с меткой.
   - Сворачивающийся блок с информацией об авторе (ФИО, группа).
   - Поле с целью ЛР.

---

## Как запустить

1. Откройте файл `calculator.html` в любом современном браузере (Chrome, Firefox, Edge и др.).
2. Калькулятор сразу готов к использованию.

> Примечание: JavaScript используется только для переключения темы. Вся стилизация и вёрстка выполнены на чистом HTML/CSS.
