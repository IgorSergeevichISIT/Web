## Структура проекта

## Проделанная работа

**1. Организация проекта**
   - **Чёткое разделение по папкам**: исходный код в `src/`, скомпилированные файлы в `dist/`
   - **Стандартная структура**: соответствие современным практикам разработки

**2. Расширение HTML-структуры**
   - Добавлены новые кнопки: `⌫`, `√`, `x²`, `x³`, `x!`, `000`
   - Добавлена вторая кнопка смены темы: "Сменить тему экрана"
   - Реорганизована сетка кнопок с использованием flexbox
   - Подключён скомпилированный JavaScript файл из папки `dist/`

**3. Реализация TypeScript логики**
   - **Основные операции**: сложение, вычитание, умножение, деление
   - **Дополнительные операции**:
     - `+/-` — смена знака числа
     - `%` — вычисление процента
     - `⌫` — удаление последней цифры
     - `√` — квадратный корень с проверкой
     - `x²`, `x³` — возведение в степень
     - `x!` — факториал (с ограничением 0-170)
     - `000` — добавление трёх нулей
   - **Обработка ошибок**: деление на ноль, переполнение, недопустимые операции
   - **Система тем**: независимое переключение темы фона и экрана

**4. Адаптивный дизайн**
   - Относительные единицы и flexbox для адаптивности
   - Ограничение максимальной ширины калькулятора

**5. Конфигурация сборки**
   - `tsconfig.json` — настройка компилятора TypeScript
   - `package.json` — управление зависимостями и скриптами

## Как запустить

**Способ 1: Готовый вариант (без установки)**
1. Убедитесь, что файл `dist/script.js` существует
2. Откройте файл `index.html` в любом современном браузере

**Способ 2: Полная сборка проекта**
1. Установите зависимости:
   ```
   npm install
   ```
2. Откройте файл `index.html` в любом современном браузере
