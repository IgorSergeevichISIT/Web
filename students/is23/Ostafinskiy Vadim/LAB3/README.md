## Структура проекта

### Корневая директория
| Файл/Папка         | Назначение                                                                 |
|---------------------|---------------------------------------------------------------------------|
| `index.html`        | Главная HTML-страница приложения                                         |
| `main.js`           | Точка входа JavaScript приложения                                        |
| `package.json`      | Метаданные проекта и зависимости                                         |
| `package-lock.json` | Фиксация версий зависимостей                                             |

### Папка `node_modules/` — зависимости
| Назначение                                                                 |
|----------------------------------------------------------------------------|
| Установленные npm-пакеты                     |

### Папка `pages/` — страницы приложения
| Файл                     | Назначение                                                                 |
|--------------------------|---------------------------------------------------------------------------|
| `main/index.js`          | Главная страница со списком пород собак                                   |
| `dog/index.js`           | Страница детальной информации о породе собаки                             |

### Папка `components/` — компоненты приложения
| Файл                          | Назначение                                                                 |
|-------------------------------|---------------------------------------------------------------------------|
| `dog-card/index.js`           | Карточка породы собаки для главной страницы                               |
| `accordion/index.js`          | Аккордеон для отображения информации на странице собаки                   |
| `back-button/index.js`        | Кнопка "Назад" для возврата на главную страницу                           |

## Проделанная работа
### **Выполнение задания по варианту 1**

1. **Инициализация проекта**
   - Инициализирован npm-проект командой `npm init`
   - Установлен Bootstrap для стилизации: `npm i bootstrap`
   - Создана базовая структура проекта с разделением на модули

2. **Создание архитектуры приложения**
   - Реализована модульная архитектура с разделением на страницы и компоненты
   - Использованы ES6 модули (`import/export`) для организации кода
   - Применён паттерн компонентного подхода для создания переиспользуемых элементов

3. **Главная страница**
   - Отображение списка карточек с породами собак
   - Динамическая генерация карточек на основе данных
   - Реализована навигация по клику на карточку
   - Использован Bootstrap для адаптивного макета


4. **Компонент карточки собаки**
   - Отображение изображения, названия породы и краткого описания
   - Уникальные идентификаторы для каждой карточки

5. **Страница детальной информации**
   - Полная информация о выбранной породе собаки
   - Изображение породы
   - Использование аккордеона для организации информации:
     - Описание породы
     - Характеристики
     - Темперамент
     - Уход и содержание
   - Кнопка возврата на главную страницу

6. **Компонент аккордеона**
   - Динамическое создание аккордеон-элементов
   - Поддержка Bootstrap Accordion API

7. **Компонент кнопки "Назад"**
   - Навигация между страницами
   - Обработка кликов для возврата на предыдущую страницу

8. **Интеграция Bootstrap**
   - Подключены CSS и JavaScript файлы Bootstrap
   - Использованы Bootstrap-компоненты: карточки, кнопки, аккордеон, сетка
   - Адаптивная вёрстка с использованием Bootstrap grid system

## Как запустить

Через Live Server
1. Установите расширение Live Server для VS Code
2. Откройте файл `index.html`
3. Нажмите кнопку "Go Live" в правом нижнем углу VS Code
