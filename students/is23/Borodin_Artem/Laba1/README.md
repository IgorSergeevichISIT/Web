# Лабораторная работа №1: Верстка калькулятора
**Студент:** Бородин Артём
**Группа:** ИС-23

## 1. Краткое описание

Данный проект представляет собой статическую верстку интерфейса калькулятора, выполненную с использованием **HTML** и **CSS**. В рамках лабораторной работы было выполнено 20 заданий по стилизации, включая создание двух цветовых тем, реализацию адаптивной сетки кнопок, центрирование элемента на странице и добавление интерактивных элементов (сворачивающиеся подробности, выпадающий список).

## 2. Содержание файлов

| Файл | Описание |
| :--- | :--- |
| `index.html` | Основной файл, содержащий HTML-структуру калькулятора. Включает все необходимые элементы для выполнения 20 заданий (кнопки, окно вывода, элементы смены темы, поле с целью ЛР). |
| `style.css` | Файл стилей, реализующий все 20 заданий. Содержит базовую темную тему, альтернативную светлую тему (`.theme-alt`). Каждая строка кода снабжена комментариями с указанием, какое задание она выполняет. |

## 3. Инструкция к запуску

Поскольку данная работа является статической версткой (HTML и CSS без логики), для ее запуска не требуется веб-сервер.

1.  **Найдите** файл `index.html` в корневой папке проекта.
2.  **Откройте** его в любом браузере.
3.  **Используйте** используйте с удовольствием.

## 4. Выполненные задания (20 пунктов)

В `style.css` реализованы следующие задания:

*   **Стилизация:** Изменение цветовой палитры, тёмный фон, квадратные кнопки, скруглённые края окна вывода, изменение шрифтов, эффекты при наведении.
*   **Расположение:** Выравнивание калькулятора по центру (Flexbox).
*   **Интерактивные элементы:** Кнопка смены темы, выпадающий список, сворачивающиеся подробности (`<details>`), подсветка слов (`<mark>`).
