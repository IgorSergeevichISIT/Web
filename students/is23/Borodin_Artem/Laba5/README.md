# Лабораторная работа №5: Веб‑приложение «VK группа» (Fetch API)

**Студент:** Бородин Артём  
**Группа:** ИС‑23  

---

## 1. Краткое описание

Лабораторная работа №5 является продолжением ЛР №4 с переходом с **XMLHttpRequest на современный Fetch API** и применением архитектурных принципов **S.O.L.I.D** для получения чистого, гибкого кода.

---

## 2. Используемые технологии

- **JavaScript ES6+**
- **Fetch API** для работы с VK API (вместо XMLHttpRequest)
- **Модульная архитектура**:
  - `modules/` — служебные модули (urls, consts)
  - `components/` — компоненты UI (Filter, UserCard, UserPage)
  - `main.js` — логика приложения

---

## 4. Функциональность приложения

**Загрузка списка участников** VK‑группы через Fetch API  
**Отображение карточек** с фото, именем и кнопкой профиля  
**Фильтрация участников** по категориям (друзья, руководители, etc)  
**Просмотр профиля** с подробной информацией (дата рождения, город, etc)  
**Возврат** на главную страницу  

---

## 5. Инструкция по запуску

### Требования
- Современный браузер (Chrome, Firefox)
- Доступ в интернет для VK API
- CORS-расширение для браузера (или использовать dev-сервер)

---

## 10. Заключение

Лабораторная работа №5 демонстрирует:

 **Практическое применение Fetch API** для работы с API  
 **S.O.L.I.D архитектура** для гибкого и поддерживаемого кода  
 **Модульная структура** с разделением ответственности  
 **Callback-функции** для инверсии зависимостей
