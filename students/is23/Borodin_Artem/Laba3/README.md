### 1. Краткое описание

Это простое двухстраничное веб-приложение, созданное в рамках лабораторной работы №3. Приложение представляет собой каталог кошек и позволяет выполнять следующие действия:

*   Просматривать список кошек на главной странице.
*   Переходить на страницу с подробной информацией о каждой кошке.
*   **Добавлять** новых кошек в каталог (путем клонирования первой кошки).
*   **Удалять** кошек из каталога.
*   Возвращаться на главную страницу с любой страницы с помощью кнопки **"Домой"** в навигационной панели.

Приложение разработано с использованием чистого JavaScript (ES6 Modules) и стилизовано с помощью **Bootstrap 5.2**, который подключается через CDN. Архитектура проекта следует принципу **"Страницы + Компоненты"**.

### 2. Инструкция по запуску

Для запуска приложения необходимо выполнить следующие шаги:

1.  Убедитесь, что у вас установлен **Node.js** и **npm**.
2.  Откройте папку с проектом в редакторе кода (например, VS Code).
3.  Запустите `live-server` в папке проекта. Он автоматически откроет `index.html` в вашем браузере по умолчанию.

После этого вы увидите главную страницу приложения с каталогом кошек.

## 3. Выполненные задания

В приложении "Каталог кошек" реализованы следующие ключевые требования и функции:

*   Архитектура Pages + Components: Приложение строго разделено на модули страниц (MainPage, CatPage) и переиспользуемые компоненты (`CatCardComponent, AlertComponent, BackButtonComponent).
*   Двухстраничное приложение: Реализована навигация между главной страницей (список кошек) и страницей профиля (детали кошки) без перезагрузки страницы.
*   Mock-объекты (коллекция): Данные о кошках хранятся в виде массива JavaScript-объектов в pages/main/index.js.
*   Bootstrap 5.2: Подключен через CDN и используется для стилизации карточек, кнопок, навигации и сетки.
*   Компонент Alert: Используется для уведомлений пользователя о действиях (например, "Кошка добавлена в избранное" или "Кошка удалена").
*   Кнопка "Добавить": На главной странице реализована кнопка, которая добавляет новую карточку кошки в коллекцию (клонируя первую карточку).
*   Кнопка "Удалить": На каждой карточке реализована кнопка, позволяющая удалить кошку из коллекции.
*   Кнопка "Домой": В навигационной панели реализована кнопка, позволяющая вернуться на главную страницу из любого места приложения.
*   Согласованность данных: Изображения и данные кошек корректно отображаются как на главной странице, так и на странице профиля.
