# Лабораторная работа №2: Реализация логики калькулятора на TypeScript

**Студент:** Бородин Артём
**Группа:** ИС-23

## 1. Краткое описание

Данный проект добавляет интерактивную логику к интерфейсу калькулятора, созданному в ЛР №1. Логика реализована на **TypeScript**. В рамках работы было выполнено 12 заданий по функционалу, включая сложные математические операции, накапливаемое сложение/вычитание и профессиональную обработку критических ошибок.

## 2. Содержание файлов

| Файл | Описание |
| :--- | :--- |
| `index.html` | Обновленный HTML-файл с версткой из ЛР №1 и добавленными кнопками для нового функционала. Содержит подключение скомпилированного скрипта `main.js`. |
| `style.css` | Файл стилей из ЛР №1, расширенный для новых кнопок. |
| `main.ts` | **Исходный код TypeScript.** Содержит класс `Calculator`, в котором реализована вся логика. Код снабжен подробными комментариями, указывающими на соответствие каждому из 12 заданий. |
| `main.js` | **Скомпилированный JavaScript.** Готовый к исполнению файл, который подключается к `index.html`. |

## 3. Инструкция к запуску

Для корректной работы калькулятора необходимо, чтобы браузер мог загрузить скомпилированный JavaScript-файл (`main.js`). Для этого требуется запуск через локальный веб-сервер.

1.  **Установите** расширение **Live Server** в Visual Studio Code.
2.  **Откройте** корневую папку проекта в VS Code.
3.  **Кликните** правой кнопкой мыши по файлу `index.html` и выберите **"Open with Live Server"** (или нажмите кнопку **"Go Live"** в правом нижнем углу VS Code).
4.  Калькулятор откроется в браузере, и весь функционал (12 заданий) будет работать.

## 4. Выполненные задания (12 пунктов)

В `main.ts` реализованы следующие функции:

*   **Базовые:** Смена знака (+/-), вычисление процента (%), стирание цифры (Backspace).
*   **Специальные:** Квадратный корень (√), возведение в квадрат (x²), факториал (x!), добавление трех нулей (000).
*   **Индивидуальная операция:** Вычисление логарифма по основанию 2 ($\log_2(x)$).
*   **Стилизация через JS:** Смена цвета фона и окна вывода по кнопке.
*   **Обработка ошибок:** Блокировка ввода после критических ошибок (`Infinity`, `Error`, `NaN`).
