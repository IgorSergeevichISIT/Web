# Лабораторная работа №1: Калькулятор (HTML/CSS)

## Цель работы
Знакомство с инструментами построения пользовательских интерфейсов веб-сайтов: HTML и CSS. Изучение структуры HTML-документа, основ верстки и стилизации элементов.

## Выполненные задачи

### 1. HTML-разметка калькулятора
Создана HTML-структура калькулятора с использованием семантических тегов:

- контейнер калькулятора
- поле вывода результата с ID "result"
- кнопки цифр (0-9) и операций
- специальные кнопки (C, +/-, %, =)

### 2. CSS-стилизация
Реализованы каскадные таблицы стилей с:

- стилями для различных типов кнопок
- эффектами наведения (hover) и нажатия (active)
- адаптивным дизайном
- центрированием элементов

## Индивидуальные доработки
По варианту выполнены следующие модификации:

1. **Изменение цветовой палитры** - применена желто-белая тема
2. **Квадратные кнопки** вместо круглых
3. **Добавление надписи** "ЛР выполнена Тихомировой Екатериной Юрьевной" в нижней части
4. **Центрирование калькулятора** на странице
5. **Увеличение размера** окна вывода
6. **Кнопка GitHub** с ссылкой на репозиторий
7. **Переключатель тем** (светлая/темная)
8. **Сворачиваемые блоки** с информацией

## Как использовать
1. Открыть файл `index.html` в браузере
2. Использовать калькулятор для базовых вычислений
3. Переключать темы с помощью соответствующей кнопки
