## Теоретическая часть

### 2.1 Инструменты для работы
Для выполнения работы использовались:
- **VS Code** с расширением **Live Server**
- **Node.js** (версия 18.x или выше)
- **npm** (Node Package Manager)

### 2.2 Основные понятия

#### Node.js
JavaScript-окружение, которое позволяет выполнять JS-код вне браузера. Используется для:
- Разработки серверной части приложений
- Сборки фронтенд-проектов
- Автоматизации задач

#### NPM (Node Package Manager)
Пакетный менеджер для управления зависимостями проекта. Основные файлы:
- `package.json` — конфигурация проекта и зависимости
- `package-lock.json` — точные версии зависимостей
- `node_modules/` — папка с установленными пакетами

#### DOM API
Набор методов для работы с HTML-документом из JavaScript:
- `getElementById()` — поиск элемента по id
- `insertAdjacentHTML()` — вставка HTML-кода
- `addEventListener()` — обработка событий

---

#### Инструкция по запуску
Для запуска моего веб-приложения нужно сделать следующее:

- **Проверка окружения**. Необходимо убедиться, что на компьютере установлены Node.js и npm.

- **Открытие проекта**. Открыть папку с проектом в VS Code или другом редакторе.

- **Запуск сервера**. Запустить live-server прямо из папки проекта.

- **Просмотр результата**. После запуска live-server автоматически откроет в браузере файл index.html, и можно будет увидеть главную страницу с каталогом продуктов.
