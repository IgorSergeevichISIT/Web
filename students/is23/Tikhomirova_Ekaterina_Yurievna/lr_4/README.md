# Лабораторная работа №4. Веб-приложение с VK API

## Информация о варианте

**Вариант:** 2  
**API метод главной страницы:** `groups.getMembers`  
**API метод страницы пользователя:** `users.get`  
**Компонент фильтрации:** `filter`  
**Описание фильтра:** Сортировка пользователей по типу

---

## Цель работы
Изучение работы с внешними API через XMLHttpRequest, получение данных из VK API и их отображение в веб-интерфейсе с реализацией фильтрации по выбранному варианту.

---

## Технологический стек

### 1. Браузерные технологии
- **HTML5** - разметка страницы
- **CSS3** + **Bootstrap 5** - стилизация
- **JavaScript (ES6+)** - логика приложения
- **XMLHttpRequest** - работа с API

### 2. Внешние сервисы
- **VK API** - получение данных о пользователях
- **Live Server** - локальный сервер для разработки

### 3. Инструменты разработки
- **VS Code** - редактор кода
- **Node.js** - среда выполнения
- **npm** - менеджер пакетов

---

## Инструкция по запуску
Откройте папку проекта в VS Code. В правом нижнем углу нажать кнопку "Go Live". Это запустит локальный сервер и автоматически откроет приложение в браузере.
Если появится ошибка CORS (когда VK API блокирует запросы), установите расширение CORS Unblock для браузера, включите его и обновите страницу.
