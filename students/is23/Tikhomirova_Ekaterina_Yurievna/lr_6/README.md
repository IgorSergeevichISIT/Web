# Лабораторная работа: Разработка серверного приложения на Node.js

## Цель работы
Освоить основы создания серверных приложений на Node.js, включая написание простого API, использование фреймворка Express и структурирование проекта.

## Выполненные этапы

### 1. Изучение Node.js
- Node.js — платформа для выполнения JavaScript-кода вне браузера, основанная на движке V8.
- Позволяет создавать серверные приложения, инструменты автоматизации и др.

### 2. Написание скриптов
- Написаны примеры скриптов для работы с файловой системой (`fs`) и вводом/выводом (`readline`).

### 3. Создание простого HTTP-сервера
- Реализован элементарный сервер с использованием встроенного модуля `http`, возвращающий текст при запросе.

### 4. Разработка API на Express
- Установлен фреймворк Express для упрощения создания сервера.
- Созданы эндпоинты:
  - `GET /stocks` — возвращает список акций.
  - `GET /stocks/:id` — возвращает акцию по ID с валидацией.

### 5. Рефакторинг и структурирование проекта
Проект организован по слоям:
- **DBConnector** — работа с файловой БД.
- **StocksRepository** — низкоуровневые операции с данными.
- **StockDAO** — высокоуровневые методы для сущности "акция".
- **StocksService** — бизнес-логика.
- **StocksController** — обработка HTTP-запросов.
- Роутинг вынесен в отдельный модуль.

### 6. Реализованные методы API
- `GET /stocks` — получение всех акций.
- `GET /stocks/:id` — получение акции по ID.
- `POST /stocks` — добавление новой акции.
- `DELETE /stocks/:id` — удаление акции по ID.

## Результат
- Создано полнофункциональное серверное приложение на Node.js с архитектурой MVC.
- API интегрировано с "файловой БД" (JSON-файл).
- Реализована обработка ошибок и валидация данных.
