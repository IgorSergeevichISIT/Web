# Лабораторная работа №2: TypeScript Калькулятор

## Цель работы
Реализация интерактивной логики калькулятора с использованием TypeScript, обработка событий и управление DOM-элементами.

## Реализованная функциональность

### Базовая логика
- Обработка цифрового ввода (0-9, точка)
- Основные операции: +, -, ×, ÷
- Вычисление результата (=)
- Очистка ввода (C)

### Дополнительные функции (по варианту)
1. **Смена знака (+/-)** - умножение на -1
2. **Проценты (%)** - деление на 100
3. **Кнопка backspace** - удаление последней цифры
4. **Смена темы** - переключатель светлой/темной темы
5. **Квадратный корень (√)** - функция Math.sqrt()
6. **Возведение в квадрат (x²)** 
7. **Факториал (x!)** - рекурсивная реализация
8. **Кнопка "000"** - добавление трех нулей
9. **Накапливаемые операции** - сохранение промежуточных результатов
10. **Смена цвета дисплея** - рандомный выбор цвета
11. **Индивидуальная операция** - возведение в куб

## Запуск проекта
1. Установите расширение **Live Server** в VS Code
2. Откройте файл `index.html`
3. Нажмите кнопку **"Go Live"**
4. Или откройте файл напрямую в браузере

## Особенности реализации
- **Локальное сохранение темы** (localStorage)
- **Валидация ввода** (предотвращение ошибок)
- **Анимации кнопок** (эффекты наведения)
- **Адаптивный дизайн** (отзывчивая верстка)
